Quick Start Setup

Ensemble AI lets multiple AI models discuss your questions together. You'll need at least one API key to get started.

What are API keys?

API keys are like passwords that let you use AI services. Each provider (OpenAI, Google, etc.) gives you your own key. You pay the provider directly for usage - we don't charge extra.

Tip: Google Gemini offers a free tier - great for trying things out!

Common issues and how to fix them:

401 - Invalid API Key

Double-check you copied the full key. Some providers require you to create a new key (you can't view existing ones).

429 - Rate Limited

You've hit usage limits. Wait a minute or check your billing/quota on the provider's dashboard.

402 - Payment Required

Your account needs credits. Add a payment method or credits on the provider's billing page.

Connection Failed

Check your internet connection. If you're on a VPN or corporate network, that might block API calls.
