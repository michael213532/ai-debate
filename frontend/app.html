Quick Start Setup

Ensemble AI lets multiple AI models discuss your questions together. You'll need at least 2 API keys to get started.

What are API keys?

API keys are like passwords that let you use AI services. Each provider (OpenAI, Google, etc.) gives you your own key. You pay the provider directly for usage - we don't charge extra.

Tip: Google Gemini offers a free tier - great for trying things out!
